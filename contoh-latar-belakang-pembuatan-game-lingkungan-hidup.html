gerak melalui keyboard input dalam game labirin menggunakan Scratch?

1. "Change x to" adalah blok kode dalam Scratch yang digunakan untuk mengubah nilai variabel x menjadi nilai baru. Dalam konteks game labirin kita bisa menggunakan blok kode ini untuk mengubah posisi karakter kita.

2. Untuk membuat karakter bergerak ke depan ke belakang ke kanan dan ke kiri dalam game labirin menggunakan Scratch kita bisa menggunakan blok-blok kode berikut:

- Untuk bergerak ke depan: Setelah mengklik bendera hijau kita bisa menggunakan blok "When green flag clicked" dan "Forever" untuk menjalankan pergerakan terus-menerus. Di dalamnya kita bisa menambahkan blok kode "If key pressed" dengan kondisi tombol panah ke atas. Jika kondisi terpenuhi maka kita bisa menggunakan blok kode "Change y by ___" dengan nilai negatif untuk membuat karakter bergerak ke atas.

- Untuk bergerak ke belakang: Kita bisa menggunakan blok kode yang sama seperti di atas tetapi dengan kondisi tombol panah ke bawah dan nilai positif dalam blok "Change y by ___".

- Untuk bergerak ke kanan: Kita bisa menggunakan blok kode "If key pressed" dengan kondisi tombol panah ke kanan. Jika kondisi terpenuhi kita bisa menggunakan blok kode "Change x by ___" dengan nilai positif untuk membuat karakter bergerak ke kanan.

- Untuk bergerak ke kiri: Kita bisa menggunakan blok kode yang sama seperti di atas tetapi dengan kondisi tombol panah ke kiri dan nilai negatif dalam blok "Change x by ___".

3. Untuk mencegah karakter melewati dinding labirin saat kita menggerakkannya ada beberapa cara yang bisa kita lakukan:

- Kita dapat menggunakan blok-blok "If" untuk memeriksa apakah karakter akan melewati dinding atau tidak. Jika dinding terdeteksi kita bisa mengubah nilai x atau y menjadi posisi sebelumnya.

- Kita juga bisa menggunakan blok-blok "Touching Color" untuk mendeteksi apakah karakter menyentuh dinding atau tidak. Jika karakter menyentuh dinding kita dapat mengubah posisi karakter sesuai dengan arah yang diinginkan.

4. Untuk membuat karakter bergerak lebih cepat atau lebih lambat dalam game labirin kita bisa mengatur nilai yang digunakan dalam blok "Change x by ___" dan "Change y by ___". Nilai yang lebih besar akan membuat karakter bergerak lebih cepat sedangkan nilai yang lebih kecil akan membuat karakter bergerak lebih lambat.

5. Untuk membuat karakter bergerak melalui keyboard input dalam game labirin menggunakan Scratch kita dapat menggunakan blok kode "When [key] pressed" untuk mendeteksi input dari keyboard. Kemudian kita dapat menambahkan blok kode "Change x by ___" dan "Change y by ___" sesuai dengan tombol yang ditekan. Misalnya jika tombol panah ke atas ditekan kita dapat menggunakan blok kode "Change y by ___" dengan nilai negatif untuk membuat karakter bergerak ke atas.

Setiap orang memiliki cara

tersendiri untuk mengusir rasa jenuh. Untuk hal tersebut,

solois Cakra Khan memilih bermain game online di sela-

sela kesibukannya.

Terlihat, saat menunggu giliran tampil sebagai pengisi

acara Tabligh Akbar di kawasan Cijantung, Jakarta

Timur, pria 26 tahun itu tengah asik

memainkan salah satu game online yang tengah populer.

"Iya kan kalau saya kalau acara kayak gini kan

kebanyakan nunggu, jadi ya ngebunuh waktunya main

game," ucapnya pada awak media.

Diakui Cakra Khan, ketertarikannya dengan permainan

virtual tersebut sudah terjadi semenjak ia duduk di

bangku sekolah. Bahkan, menurutnya, demi bermain

game, semasa sekolah dulu ia rela untuk membolos.

"Saya dari kecil jamannya dulu di kampung saya,

jamannya (game) ding dong itu emang saya bela-belain

bolos sekolah buat main itu. Jadi ketika saya bisa beli

setiap game, apapun saya coba," aku Cakra Khan sambil

tertawa.

Kegandrungan Cakra Khan pada game pun menjadi

salah satu keuntungan saat memasuki bulan ramadan

seperti sekarang. Guna membunuh waktu, pria asal

Pangandaran itu pun mengaku sampai menghabiskan

waktu berjam-jam untuk sekedar menjalani hobinya

tersebut.

"Iya karena memang kalau sudah addicted ke sebuah

permainan kan kalau kalah nyoba lagi, jadi nggak lihat

jam, memang harus ada yang ngalihin perhatian. Kalau

sore dari jam 3 nih, kalau malam bisa dari jam 12

sampai sahur," paparnya

Dikatakan Cakra Khan, kecintaannya pada game tak

hanya dari gadgetnya yang dipenuhi berbagai jenis

permainan. Dikediamannya pun, ia memiliki console

game yang selalu ia mainkan saat tidak memiliki

kesibukan.

"Console (game) punya di rumah. Tapi kalau nunggu

kayak gini nggak akan bisa bawa yang gede (console

game) jadi di gadget aja," tuturnya.

Bahkan, pelantun lagu Kekasih Bayangan itu juga

menambahkan, dirinya sampai harus menyiapkan budget

khusus untuk mengakomodir segala keperluan terkait

hobinya bermain game.

"Budget khusus itu udah disiapin, jadi maksudnya sudah

ada. Karena kan kalau console dari satu game beres

nggak mainin lagi, jadi beli kaset baru lagi. Tapi kalau

yang di gadget kan muter-muter terus, cuma tetep kartu

kredit jadi sasaran," pungkas Cakra Khan.
