1. Sebuah ayunan bandul sederhana dalam waktu 20 sekon mengalami 10 ayunan. Periodedan frekuensi avunan adalah​


a. periode getaran adalah [tex]\boxed{\bold{2 \: s}}[/tex]

b. frekuensi getaran adalah [tex]\boxed{\bold{0,5 \: Hz}}[/tex]

Pembahasan

Hi! soal tersebut adalah membahas mengenai getaran dan gelombang. Singkatnya, getaran adalah pergerakan berbolak balik di sekitar keseimbangan. Biasanya getaran di ilustrasikan dengan bandul atau gerakan pegas.

Rumus menentukan frekuensi :

[tex]\boxed{\bold{f = \frac{n}{t}}}[/tex]

Dimana :

[tex]\bold{f}[/tex] = frekuensi.

[tex]\bold{n}[/tex] = jumlah getaran dalam waktu t sekon.

Berarti :

[tex]\boxed{\bold{f = \frac{1}{T}}}[/tex]

Rumus menentukan periode :

[tex]\boxed{\bold{T = \frac{t}{n}}}[/tex]

Dimana :

[tex]\bold{T}[/tex] = periode.

[tex]\bold{n}[/tex] = banyak getaran.

[tex]\bold{t}[/tex] = waktu.

Berarti :

[tex]\boxed{\bold{T= \frac{1}{f}}}[/tex]

Langkah Penyelesaian

Diketahui :

Waktu = 20 s.

Banyak getaran = 10 kali.

Ditanya :

a. Periode getaran ?

b. Frekuensi getaran?

Dijawab :

Mencari periode :

[tex]\bold{T = \frac{t}{n}}[/tex]

[tex]\bold{T = \frac{20}{10}}[/tex]

[tex]\bold{T = 2 \: s}[/tex]

Mencari frekuensi :

[tex]\bold{f = \frac{n}{t}}[/tex]

[tex]\bold{f = \frac{10}{20}}[/tex]

[tex]\bold{f = 0,5 \: Hz}[/tex]

Kesimpulan :

a. periode getaran adalah [tex]\boxed{\bold{2 \: s}}[/tex]

b. frekuensi getaran adalah [tex]\boxed{\bold{0,5 \: Hz}}[/tex]

5. Jika ayunan sederhana bergetar sebanyak 225 kali dalam waktu ¼ jam, maka trntukanlah Frekuensi dan Periode ayunan tersebut!​


Jawaban:

Rumus frekuensi:

f = n / t = 1 / f

Rumus periode:

T = t / n = 1 / f

Keterangan:

f = frekuensi (Hz)

T = periode (s / detik)

n = Jumlah Getaran

t = Waktu (s / detik)

Diketahui:

n = 225 kali

t = 1/4 jam = 15 menit = 900 detik

Ditanya:

a. f (frekuensi) = ?

b. T (periode) = ?

Jawab:

a. f = n / t

f = 225 / 900 detik

f = 1/4 Hz = 0.25 Hz

b. T = 1 / f

T = 1 / 0.25 Hz

T = 4 detik

Jadi, Frekeunsi pada ayunan tersebut adalah 0.25 Hz, dan Periode ayunan tersebut adalah 4 detik.

11. Jika ayunan sederhana bergetar sebanyak 50 kali dalam waktu 10 sekon. Maka besar frekuensi ayunan dan periode ayunannya adalah...


Jika ayunan sederhana bergetar sebanyak 50 kali dalam waktu 10 sekon , maka besar frekuensi ayunan dan periode ayunannya adalah 5 Hz dan 0,2 sekon

PEMBAHASAN

Gelombang sinusoidal adalah gelombang yang membentuk fungsi sinus. Beberapa rumus dasar gelombang yang perlu di ingat adalah sebagai berikut ini :

f = 1 / T

v = λ . f

Δx / λ = ΔΘ / 2π

Δt / T = ΔΘ / 2π

dimana :

f = frekuensi gelombang ( Hz )

T = periode gelombang ( s )

v = cepat rambat gelombang ( m/s )

λ = panjang gelombang ( m )

Δx = jarak pisah antar dua titik pada gelombang ( m )

Δt = selang waktu ( s )

ΔΘ = beda fase ( radian )

Bentuk persaman gelombang adalah :

y = A sin ( ωt - kx )

dimana :

y = simpangan gelombang ( m )

A = amplitudo gelombang ( m )

ω = frekuensi sudut ( rad/s )

t = waktu ( s )

k = bilangan gelombang ( 1/m ) = 2π/λ

x = jarak titik dari sumber ( m )

Okay mari kita gunakan rumus ini untuk menyelesaikan soalnya.

Diketahui :

Jumlah getaran = n = 50 kali

Waktu getaran = t = 10 sekon

Ditanyakan :

f = ?

T = ?

Penyelesaian :

f = n / t

f = 50 / 10

f = 5 Hz

T = t / n

T = 10 / 50

T = 0,2 sekon

26. Jika ayunan sederhana bergetar sebanyak 60 kali dalam waktu 15 sekon tentukan frekuensi ayunan dan periode ayunan


Jawaban:

frekuensi ayunan sederhana tersebut adalah 4 Hz dan Periode ayunan tersebut adalah  0,25 sekon

Penjelasan:

Diketahui :

n = 60 Kali t = 15 sekon f = ...??T = ...???

Ditanya :

tentukan frekuensi ayunan dan periode ayunan

Jawab

Kita akan mencari yaitu frekuensi terlebih dahulu, maka dirumuskan dengan

f = n / t f = 1 / T

Kemudian kita masukkan rumus tersebut, Jika diketahui Jumlah getaran dan waktu (sekon)

f = n / t f = 60 / 15 f = 4 Hz

Kemudian setelah kita menemukan frekeunsi, maka kita akan mencari yaitu periode, dengan rumus

T = t / n T = 1 / f

Kemudian kita masukkan rumus tersebut, Jika frekeunsi sudah diketahui

T = 1 / f T = 1 / 4 T = 0,25 s

Kesimpulan yang didapat adalah

Jadi, frekuensi ayunan sederhana tersebut adalah 4 Hz dan Periode ayunan tersebut adalah  0,25 sekon

Pergerakan bandul :

Jika bandul A-B = [tex]\bold{\frac{1}{4}}[/tex] getaran. Jika bandul A-B-C = [tex]\bold{\frac{1}{2}}[/tex] getaran. Jika bandul A-B-C-B = [tex]\bold{\frac{3}{4}}[/tex] getaran. Jika bandul A-B-C-B-A = [tex]\bold{1}[/tex] getaran.

Rumus menentukan frekuensi :

[tex]\boxed{\bold{f = \frac{n}{t}}}[/tex]

Dimana :

[tex]\bold{f}[/tex] = frekuensi. [tex]\bold{n}[/tex] = jumlah getaran dalam waktu t sekon.

Berarti :

[tex]\boxed{\bold{f = \frac{1}{T}}}[/tex]

Rumus menentukan periode :

[tex]\boxed{\bold{T = \frac{t}{n}}}[/tex]

Dimana :

[tex]\bold{T}[/tex] = periode. [tex]\bold{t}[/tex] = waktu. [tex]\bold{n}[/tex] = banyak getaran.

Berarti :

[tex]\boxed{\bold{T = \frac{1}{n}}}[/tex]

Langkah Penyelesaian

Diket :

Banyak getaran = 30 kali. Waktu = 8 s.

Ditanya :

Frekuensi dan periode?

Dijawab :

Frekuensi nya adalah

[tex]\bold{f = \frac{n}{t}}[/tex]

[tex]\bold{f = \frac{30}{8}}[/tex]

[tex]\bold{f = 3,75 \: Hz}[/tex]

Periode nya adalah

[tex]\bold{T = \frac{t}{n}}[/tex]

[tex]\bold{T = \frac{8}{30}}[/tex]

[tex]\bold{T = 0,266 \: s}[/tex]

Kesimpulan

Jadi frekuensi ayunan adalah [tex]\boxed{\bold{3,75 \: Hz}}[/tex] sedangkan periode ayunan adalah [tex]\boxed{\bold{0,266 \: s}}[/tex]

n = banyak getaran

A = Amplitudo  

t = waktu (s)  

s = jarak gelombang (m)
