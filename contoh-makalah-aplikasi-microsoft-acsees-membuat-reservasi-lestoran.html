10. 1. Program aplikasi yang digunakan untuk pengolahan kata seperti membuat makalah, surat, skripsi disebut....a. wordartb. microsoft excel c. wordpard d. microsoft word e. microsoft power point​


Aplikasi yang digunakan dalam pengolahan kata seperti membuat makalah, surat, skripsi dinamakan d. microsoft word. Microsoft Word adalah program aplikasi dari microsoft office yang sering dipakai untuk pengelolahan teks, pengelolahan dokumen, laporan dan lain-lain.

PEMBAHASAN

Microsoft Word

Microsoft Word merupakan program pengolah kata berupa software aplikasi yang fungsinya untuk membuat tulisan dengan berbagai macam format, seperti surat, laporan, jurnal, dan sebagainya.

Namun, saat ini Microsoft Word tidak cuma dapat digunakan untuk mengolah kata, melainkan juga bisa untuk menyisipkan gambar, tabel, atau grafik, sehingga dapat menampilkan tulisan jadi lebih informatif.

Wordart

WordArt merupakan fitur pada Microsoft Word yang berfungsi dalam memberi efek khusus pada teks atau tulisan. Efek pada tulisan itu bisa seperti efek shadow (bayangan), reflection (cermin), glow, bevel, transform, 3 dimensi (3D) dan sebagainya.

Fungsi WordArt yaitu membuat teks atau tulisan dengan gaya artistik. Disamping itu, WordArt kita juga dapat memutar tulisan vertikal ataupun membuat teks terbalik pada Microsoft Word.

Wordpad

WordPad merupakan pengolah kata dasar yang ada di setiap versi sistem operasi Microsoft sejak Windows 95. WordPad bisa digunakan dalam membuat dan memodifikasi dokumen.

Meskipun kinerjanya lebih lambat daripada Notepad, WordPad  bisa menangani grafik dan pemformatan kaya. Hal inilah yang membuat WordPad lebih disukai untuk membuat catatan cepat dan melakukan kegiatan menulis berbasis teks.

Microsoft power point

Power Point adalah salah satu aplikasi dari microsoft yang ditujukan sebagai media presentasi.

11. bagaimana cara membuat aplikasi microsoft excel?


Masuk Ms. Excel dan buat 3 buah sheet dengan masing masing nama sheet "Utama, Main, Stock, dan Laba". Untuk sheet utama jangan diisi data apapun namun beri warna background agar lebih menarik.

Pada sheet Main, buat tabel transaksi seperti terlihat pada gambar berikut

Pada cells F7 ketikan =IF(C7="","",VLOOKUP(C7,stock!$B$4:$C$23,2,FALSE))
pada cell G7 ketikan =IF(D7="","",VLOOKUP(C7,stock!$B$4:$F$24,3,FALSE))
pada cells H7 ketikkan =IF(E7="","",VLOOKUP(C7,stock!$B$4:$E$23,4,FALSE))
pada cells I7 ketikkan =IF(D7="","",G7*D7+H7*E7)
pada cells J7 ketikan =IF( E7="","",IF(K7-D7-E7*L7<0,"stock kurang","ada"))
pada cells I20 ketikkan =IF(I7="","",SUM(I7:I19)) dan
pada cell I22 ketikkan =IF(I20="","",I21-I20)

Pada sheets Stock buat database barang seperti berikut
Pada sheet Laba buat tabel seperti pada gambar berikut

Setelah tabel laba terbuat ketikan rumus di cell C3 berikut=IF(main!C7="","",VLOOKUP(main!C7,stock!$B$4:$H$23,7,FALSE)) dan di cells D3 ketikan =IF(main!I7="","",(main!I7-((main!D7)+(main!E7*main!L7))*C3))
